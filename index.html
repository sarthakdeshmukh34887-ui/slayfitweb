How to install

I do not host this app on the official play store. Here is the process.

01
Download the APK file

Tap the button. It is 70 MB so give it a second on slow networks.

02
Allow unknown sources in settings

Go to Settings, Security, then Unknown Sources. On Android 8, it is per-app.

03
Open the file, hit install

Locate it in files. Enter download manager. Tap and complete.

04
Log your first workout

Skip account setup if preferred. Start tracking. Metrics optimize soon.
